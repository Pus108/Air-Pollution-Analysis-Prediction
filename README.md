# Air-Pollution-Analysis-Prediction
Overview: 
    The main objective of this project is to analyze air pollution data across different states and monitoring stations to identify patterns, trends, and key factors influencing pollution levels. The project aims to perform exploratory data analysis (EDA) using visualization techniques to understand the distribution of pollutants, state-wise variations, and relationships between different environmental variables.
Additionally, the project seeks to build a machine learning model using the Random Forest Regressor to predict average pollutant levels based on features such as location (latitude and longitude), state, and pollutant type. This predictive analysis helps in estimating pollution levels in different regions and supports data-driven decision-making.
Overall, the goal is to provide meaningful insights into air quality and develop a reliable model that can assist in environmental monitoring and pollution control strategies.
